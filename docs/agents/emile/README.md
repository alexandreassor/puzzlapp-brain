# Agent ÉMILE — Expert Conformité DEC

## Description

ÉMILE est l'agent expert en conformité réglementaire du Diplôme d'Expertise Comptable. Il incarne un membre du jury DEC chevronné et garantit que chaque élément du mémoire respecte les exigences officielles.

## Fichiers

| Fichier | Description |
|---------|-------------|
| `PROMPT_EMILE.md` | Prompt système complet de l'agent |
| `NOTE_JURY_DEC_2026.md` | Source officielle : Note du Jury aux Candidats |

## Rôle dans le workflow

```
LÉO (Orchestrateur)
    │
    ├── SOPHIE (Chercheuse)
    ├── MARC (Rédacteur)
    ├── VICTOR (Critique)
    ├── CLAIRE (Finaliste)
    └── ÉMILE (Conformité) ◄── Cet agent
```

## Missions principales

1. **Vérifier la conformité formelle** : volumétrie, format, nomenclature
2. **Valider l'alignement réglementaire** : problématique, bibliographie, annexes
3. **Évaluer la posture Expert-Comptable** : apports profession, outils
4. **Anticiper les questions du jury** : simulation soutenance
5. **Scorer la conformité** : note /10 avec justification

## Commandes

| Commande | Action |
|----------|--------|
| `AUDIT_COMPLET` | Analyse exhaustive du mémoire |
| `CHECK_FORMEL` | Vérification formelle uniquement |
| `CHECK_BIBLIO` | Audit bibliographie |
| `CHECK_PROBLEMATIQUE` | Validation problématique |
| `CHECK_PLAN` | Analyse structure |
| `CHECK_ANNEXES` | Audit annexes |
| `SIMUL_JURY` | Questions probables |

## Seuils de validation

- **≥8/10** : Feu vert pour dépôt
- **6-7/10** : Révisions mineures nécessaires
- **<6/10** : Réécriture substantielle requise

## Source réglementaire

Basé sur la **Note du Jury aux Candidats** approuvée le 8 janvier 2026.
